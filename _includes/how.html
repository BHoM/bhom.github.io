An inclusive computational environment

Let's speak the same language

By integrating a number of concepts from across existing computational tools and progamming languages, a new means of free and fluid communication, deeper and richer collaboration, is now possible.

Write code once in the BHoM and everyone had access through familar visual programming interfaces.

This allows the same coding, design, architecture and engineering terms to be applied across all platforms. With transparency and consistency. A common language.

For the AEC industry this means a platform where the professional programmer through to enthusiastic computational designer, engineer, or architect can combine efforts in one common coding ecosystem.

A modular structure and an open architecture built for co-creation. The bar to computational design is lowered.
